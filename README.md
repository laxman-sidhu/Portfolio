## рҹ§© Edit skills, experience & education
All three live in **`js/config/home-config.js`** вҖ” no HTML needed.

- **Skills** (`window.SKILLS`): each `{ }` is a category. Add a category by copying
  a block; add skills inside one by adding strings to its `items`. `tone` picks the
  pastel colour (blush В· rose В· peach В· butter В· sage В· mint В· sky В· lavender В· lilac).
- **Work experience** (`window.EXPERIENCE`): currently dummy data вҖ” replace with real
  roles (`period`, `role`, `company`, `place`, `desc`, `tags`). Set it to `[]` to hide
  the whole section.
- **Education** (`window.EDUCATION`): `period`, `title`, `place`, optional `score`/`desc`.

## вһ• Add a project (no HTML needed)
Open **`js/config/projects-config.js`** and add a block to `PROJECTS_MAIN`
(or `PROJECTS_FUN`):

```js
{
  name: "My Project",
  thumbnail: "assets/Projects/my-thumb.png",
  desc: "One line about it.",
  tags: ["Python", "SQL"],
  difficulty: "intermediate",   // easy | intermediate | difficult
  code: "https://github.com/...",  // or "None"
  demo: "https://...",             // or "None"
}
```
Button rule (automatic):
- `demo` is a real link вҶ’ shows **Live Demo** + **Code**
- `demo` is `"None"` вҶ’ shows only **View Code** (full width)
- `code` is `"None"` too вҶ’ that button is hidden as well

## рҹ–јпёҸ Add hobby photos / videos
1. Drop files into the matching folder, e.g. `assets/Hobbies/Dance`.
   (Supported: jpg, jpeg, png, gif, webp, mp4, webm, mov, ogg)
2. **Double-click `update-hobbies.bat`** вҖ” it rewrites `js/config/hobbies-media.js`
   with the new file names automatically.
3. Refresh `hobbies.html`. Done.

To add a whole new hobby: create a folder `assets/Hobbies/<Name>`, add a block
in `js/config/hobbies-config.js` with `folder:"<Name>"`, then run the .bat.

## вңүпёҸ Contact form
Wired to your EmailJS account in `js/main/contact.js` (public key + service +
template already filled in). It sends `from_name`, `from_contact`, and
`message`.
